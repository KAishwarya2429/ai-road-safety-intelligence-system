# 🚦 AI Road Safety Intelligence System

An AI-powered web application that identifies accident-prone areas and provides real-time safety recommendations using intelligent analysis and geospatial visualization.

---

## 🌍 Problem Statement

Road accidents are a major concern in India due to:
- High traffic density
- Poor road conditions
- Lack of real-time safety insights

---

## 💡 Solution

This project uses AI + data simulation to:
- Predict high-risk road zones
- Provide safety recommendations
- Visualize locations on maps

---

## 🚀 Features

✅ AI-powered safety insights  
✅ Risk classification (High / Medium / Low)  
✅ Smart suggestions for safer travel  
✅ Interactive map visualization  
✅ Fast and responsive UI  

---

## 🧠 Tech Stack

- Frontend: React + Vite  
- Backend: Node.js + Express  
- AI: Gemini API (with fallback logic)  
- Maps: Google Maps Embed API  
- Data: Mock dataset (simulation-based)

---

## 📊 Demo Output

- 📍 Shivajinagar → High Risk  
- 📍 Katraj → Medium Risk  
- 📍 Hinjewadi → Low Risk  

With AI-generated safety insights and route suggestions.
